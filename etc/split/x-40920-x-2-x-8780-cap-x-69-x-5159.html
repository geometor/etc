As in the preamble just before X(24307), the obverse triangle, A'B'C', of a point P = p : q : r is given by

A' = p : r : q, B' = r : q : p, C' = q : p : r.

In particular, for P = X(69),

A' = b^2 + c^2 - a^2 : a^2 + b^2 - c^2 : c^2 + a^2 - b^2.
