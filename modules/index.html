geometor.etc¶

Tools for processing the Encyclopedia of Triangle Centers (ETC).

Key Components:¶

  • Ingest: Tools (ingest_folder()) to split and organize the raw HTML source.

  • Parser: Tools (parse_center()) to extract center data from HTML.

  • Transform: Tools (generate_rst_files()) to generate ReStructuredText documentation.

  • Glossary: Tools (extract_glossary_terms()) to handle glossary terms.

Usage:¶

Use the main function in __main__.py or the individual modules to process ETC data.
